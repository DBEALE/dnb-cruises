    What do the class dots mean?

    The dots next to each ship class are a 1–5 rating of how modern / flagship-class the ship is.

    • Newest flagship — Icon
    • Modern flagship — Oasis, Quantum, Edge, Prima, Breakaway Plus
    • Recent generation — Freedom, Solstice, Breakaway, Epic
    • Older generation — Radiance, Voyager, Millennium, Jewel, Spirit, Dawn, America
    • Legacy class — Vision, Sun, Galapagos
